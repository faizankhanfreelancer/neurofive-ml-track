# Titanic Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The goal is to understand the dataset, identify missing values, explore feature distributions, and discover insights before building machine learning models.

---

## 📂 Dataset

- Titanic - Machine Learning from Disaster
- Source: Kaggle

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed

- Dataset Overview
- Missing Value Analysis
- Data Types
- Statistical Summary
- Numerical vs Categorical Features
- Survival Analysis
- Passenger Class Analysis
- Gender Analysis
- Correlation Heatmap
- Age & Fare Distribution

---

## 📈 Key Findings

- Female passengers had a higher survival rate.
- First-class passengers survived more frequently.
- Cabin contains many missing values.
- Age and Fare influence passenger characteristics.
